### Integers

**unsigned:** can only be positive
| length | signed | unsigned |

| 16-bit | i16    | u8       |
| 32-bit | i32    | u8       |
| 64-bit | i64    | u8       |
| 128-bit | i128  | u8       |
| arch | isize    | usize    |
